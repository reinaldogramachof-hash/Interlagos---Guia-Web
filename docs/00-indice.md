## Resumo Executivo

**Produto:** Plataforma digital comunitária hiperlocal — rede social do bairro.
**Primeiro bairro:** Parque Interlagos, São José dos Campos — SP.
**Lançamento:** Maio/2026.
**Equipe:** 3 pessoas (Dev/Gestor + Especialista Digital + Marketing).
**Modelo:** Freemium B2B — planos mensais canceláveis (R$0 / R$19,90 / R$39,90 / R$79,90 / R$299,90).
**Stack:** React 19 + Vite 7 + Supabase + Tailwind + PWA.
**Meta de lançamento:** 1/3 dos comércios do bairro cadastrados.
